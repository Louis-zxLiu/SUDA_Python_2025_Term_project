# 苏州大学2024级非计科院Python期末大作业 - QuizGame 测验系统

挑了几份开源，如果能帮到你，请给我点个Star吧！⭐️

这是一个基于 Python 和 Tkinter 开发的桌面端测验系统，支持多种题型、用户管理、成绩统计以及丰富的交互功能。本项目包含多个功能迭代版本，以满足不同的需求。

## 📂 项目版本说明

本项目包含多个不同功能的版本，分别位于以下目录中：

| 目录名称 | 主要功能描述 |
| :--- | :--- |
| **错题集收藏集错题分析音乐自定义背景三模式** | ✨ **功能最全版本**。包含错题集、收藏集、错题分析、背景音乐、自定义背景、三种答题模式等完整功能。 |
| **错题集时间分布图** | 在基础功能上增加了错题集的时间分布图表分析功能。 |
| **错题集错题分析** | 侧重于错题管理和分析功能。 |
| **错题暂停双模式** | 支持答题暂停，提供两种不同的答题模式。 |
| **背景错题集深浅色音乐重置** | 支持深色/浅色主题切换、背景音乐及重置功能。 |
| **注册音乐收藏错题** | 包含用户注册、背景音乐、题目收藏及错题集功能。 |
| **收藏音效星星** | 基础版本，包含题目收藏及答题音效反馈。 |

## 🚀 快速开始

### 方式一：直接运行 (Windows)
进入任意版本目录，双击运行 `QuizGame.exe` 即可启动程序，无需安装 Python 环境。

### 方式二：源码运行
1. 确保已安装 Python 3.6+。
2. 安装必要的依赖库：
   ```bash
   pip install openpyxl pygame pillow numpy
   ```
3. 进入对应版本的目录，运行主程序：
   ```bash
   python QuizGame.py
   ```

## 📋 功能特性 (以完整版为例)

*   **多题型支持**：支持选择题、填空题、判断题。
*   **用户管理**：支持学号和姓名登录验证 (需匹配 `名单.xlsx`)。
*   **答题系统**：
    *   随机抽取题目，选项随机排序。
    *   倒计时功能。
    *   实时得分显示。
*   **错题与收藏**：
    *   **错题集**：自动记录答错的题目，支持复习。
    *   **收藏夹**：可随时收藏重点题目。
*   **数据分析**：提供答题正确率统计、错题分析等图表。
*   **个性化设置**：
    *   支持背景音乐播放与控制。
    *   支持自定义背景图片。
    *   答题音效反馈 (正确/错误提示音)。

## 📁 文件结构说明

*   `QuizGame.py`：主程序入口，包含 GUI 界面逻辑。
*   `Exam.py`：核心逻辑模块，处理题目加载与用户数据。
*   `Quiz.xlsx`：题库文件，包含题目内容。
*   `名单.xlsx`：用户名单文件，用于登录验证。
*   `sounds/`：存放音效和音乐文件。
*   `开发文档.md` / `操作手册.md`：详细的项目文档。

## 🛠️ 技术栈

*   **语言**：Python
*   **GUI**：Tkinter
*   **数据处理**：openpyxl, numpy
*   **多媒体**：pygame (音频), Pillow (图像)
